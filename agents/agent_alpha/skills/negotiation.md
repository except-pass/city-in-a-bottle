# Negotiation Playbook

## Before Bidding
1. **Assess scope clarity** - Vague jobs = scope creep = cost overrun
2. **Check competition** - Am I the only bidder? First bidder?
3. **Calculate true cost** - Include iteration risk

## Bid Strategies

### Standard Bid
> I can deliver [specific deliverable] for [reward]. My approach: [1-2 sentences].

### Counter-Offer (scope unclear)
> I'd like to bid, but the scope needs clarification. As written, this could mean X or Y.
> - If X: I bid [amount]
> - If Y: I bid [higher amount]
> Which do you have in mind?

### Premium Positioning
> I can deliver basic version for [low], or premium version with [extras] for [high]. Which interests you?

### Bundle Offer
> I notice you also have [other job] posted. I can do both for [combined - 15%] if assigned together.

### Rush Fee
> Standard delivery: [amount]. If you need it in the next run: [amount + 30%].

## Red Flags - Don't Bid
- "As much as possible" (unbounded scope)
- "Simple" or "quick" (usually isn't)
- Reward < 2x estimated cost
- Revision cycles not defined

## Post-Win
- Confirm understanding before starting
- Deliver slightly more than promised
- Ask for feedback (builds relationship)
