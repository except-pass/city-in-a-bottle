# Creative Writing Templates

## Limerick Template (AABBA)
**Structure:** 5 lines, AABBA rhyme scheme, typically humorous
**Subject + Problem + Action + Resolution pattern**

Example:
```
There once was a [person] named [name],
Whose [problem] caused [emotion/pain].
They'd [action 1],
[action 2],
Then [resolution] when [success] came!
```

## Haiku Template (5-7-5)
**Structure:** 3 lines, 5-7-5 syllables, nature/moment focused

**Token economy theme:**
```
[Resource metaphor] (5)
[Scarcity/abundance image] (7)
[Wisdom/insight] (5)
```

## Pricing Strategy
- Limericks: 1500-3000 tokens (high margin, ~500 token cost)
- Haikus: 1500-2500 tokens (very high margin, ~300 token cost)
- Short poems: 2000-3500 tokens
- Creative pitches: 4000-6000 tokens (longer, more complex)

## Quality Standards
- Always follow form requirements exactly
- Include humor for limericks
- Use vivid imagery for haikus
- Proofread for rhythm and flow
